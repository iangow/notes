Working with DERA Notes data using Polars

The source code for this note can be found here. The package era_pl is installed using pip install 'era-py[polars]'. See here for a discussion of how to get the DERA Notes data into Parquet files.

import polars as pl
from era_pl import load_parquet

There are three Parquet files that I use here:

  • sub_notes_2024q1
  • num_notes_2024q1
  • dim_notes_2024q1
submissions = load_parquet('sub_notes_2024q1', 'dera_notes')
numericFacts = load_parquet('num_notes_2024q1', 'dera_notes')
dimensions = load_parquet('dim_notes_2024q1', 'dera_notes')

The commands above are equivalent to the following if you put the data files in your working directory:

submissions = pl.scan_parquet('sub_notes_2024q1.parquet')
numericFacts = pl.scan_parquet('num_notes_2024q1.parquet')
dimensions = pl.scan_parquet('dim_notes_2024q1.parquet')

I use the former form because I have organized my Parquet data repository along the lines outlined here or here.

    submissions
